## 🎯 界面功能区域说明

### 1. 顶部导航栏
- **项目标题**: AI Image Tree System
- **语言切换**: 支持中英文切换
- **系统设置**: AI提供商配置、质量控制等

### 2. 创意输入区
- **提示词输入框**: 输入创意描述
- **生成按钮**: 启动创意树生成
- **最近创作**: 快速访问历史创作

### 3. 树状可视化区域
- **根节点**: 显示原始提示词和生成的图像
- **分支节点**: 基于关键词生成的创意分支
- **连接线**: 清晰展示节点间的层级关系
- **交互控制**: 节点展开/收缩、图像缩放

### 4. 关键词提取面板
- **自动提取**: AI智能分析提示词
- **关键词列表**: 可选择的创意方向
- **批量操作**: 一键生成所有分支

### 5. 图像操作功能
- **全屏查看**: 点击图像进入全屏模式
- **缩放控制**: 鼠标滚轮缩放图像
- **拖拽平移**: 缩放状态下拖拽查看
- **四格对比**: 同级节点对比查看

### 6. 工具栏功能
- **树状态显示**: 当前创作的基本信息
- **操作按钮**: 保存、导出、分享等
- **进度指示**: 任务执行状态显示

## 🎯 使用场景展示

### 创意设计师工作流
1. **输入创意概念** → 在输入框描述设计需求
2. **AI分析提取** → 系统自动提取关键创意元素
3. **选择方向** → 从关键词中选择感兴趣的方向
4. **生成分支** → 系统生成多个创意变体
5. **深度探索** → 继续从任意分支扩展新创意
6. **对比选择** → 使用四格对比功能选择最佳方案

### 商业用户应用
1. **品牌概念输入** → 描述品牌定位和视觉需求
2. **多方案生成** → 快速生成多种视觉风格
3. **团队协作** → 分享链接进行团队讨论
4. **迭代优化** → 基于反馈继续细化创意
5. **最终确定** → 导出高质量图像用于实际应用

- **按需刷新**: 只在有任务时才刷新数据
