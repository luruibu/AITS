## 🎯 核心交互流程

### 创意生成流程
1. **输入提示词** → "现代科技公司logo设计"
2. **AI分析** → 提取关键词：现代、科技、简约、蓝色
3. **生成根节点** → 创建初始图像
4. **选择关键词** → 选择感兴趣的创意方向
5. **生成分支** → 基于关键词创建4个分支
6. **继续探索** → 从任意分支继续扩展

### 图像操作流程
1. **点击图像** → 进入全屏查看模式
2. **鼠标滚轮** → 缩放图像（50%-500%）
3. **拖拽移动** → 查看图像细节
4. **双击重置** → 恢复原始大小
5. **ESC退出** → 返回树状视图
